RESPONSIVE BREAKPOINTS
═══════════════════════════════════════════════════════════════

Mobile (< 768px):
├─ Stats cards: 1 column stack
├─ Chart + Activity: 1 column stack
└─ Thread cards: 1 column

Tablet (768px - 1024px):
├─ Stats cards: 3 columns
├─ Chart + Activity: 1 column stack
└─ Thread cards: 2 columns

Desktop (> 1024px):
├─ Stats cards: 3 columns
├─ Chart + Activity: 2-1 column split
└─ Thread cards: 2 columns


THEME SUPPORT
═══════════════════════════════════════════════════════════════

Light Theme:
├─ Background: #FFFFFF
├─ Cards: #FFFFFF
├─ Text: #000000
├─ Borders: #E5E5E5
└─ Chart grid: Light gray

Dark Theme:
├─ Background: #000000
├─ Cards: #14213D (navy blue)
├─ Text: #FFFFFF
├─ Borders: Dark gray
└─ Chart grid: Dark gray

All components use CSS variables (--background, --card, etc.)
for automatic theme switching via Tailwind's dark mode.
